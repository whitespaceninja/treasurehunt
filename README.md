# treasurehunt
simple javascript console game

HOW TO PLAY

1) Install node
2) Run "node treasure-hunt.js"
3) Click ENTER
4) Use w, a, s, d to move your character towards the $ sign

| Character | Control   | 
| 'w'       | UP        |
| 'd'       | RIGHT     |
| 's'       | DOWN      |
| 'a'       | LEFT      |
| 'c'       | Quit      |
